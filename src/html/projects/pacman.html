Movement & animation

I used a 28 x 30 grid to create the board.
I used a two-dimensionnal array to translate the positions of the walls, dots, ghosts and pac-man.
This enabled me to test where characters could go.

I created an animation function that gets called by a requestAnimationFrame(), that translates the character in the right direction by x percent(I used x to change the speed).

Every time the translation is at/over 100% a move() function is called that deletes the character from its tile and appends it to the next one.

Object oriented programming

All the characters where calling the same move() and animation() functions, so they were interrupting each other.
To solve this issue, I re-wrote the program in OOP, using the ES2015 “class” syntax. It saved me lines of code by creating a Character class which the other classes extend and also made it easier to add different speeds, states...
